Pollinator Quest

About the Game

Pollinator Quest is a relaxing yet strategic adventure game where players restore pollinator paths across urban cities by planting flowers and guiding bees. Built for the Green Game Jam to raise awareness on declining pollinator populations.

Eco-Friendly Features

  • Highlights native plants and pollinator habitats.
  • Teaches about ecological networks through gameplay.
  • Uses minimal animation to reduce processing power.

Quick Info

  • Genre: Eco Adventure
  • Built With: Godot
  • Platform: Mobile & Web
  • Team: Nectar Crew
