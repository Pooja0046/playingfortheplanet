Ocean Guardian

About the Game

Ocean Guardian is an adventure puzzle game where players clean up plastic pollution, rescue marine animals, and restore coral reefs. Developed for the Green Game Jam, it highlights ocean conservation efforts in a fun, engaging way.

Eco-Friendly Features

  • Players learn how pollution affects marine life in real-time.
  • Rewards are earned by planting virtual coral that mirrors real-world reforestation donations.
  • Designed with low-energy graphics and optimized loading for eco-mode users.

Quick Info

  • Genre: Adventure Puzzle
  • Built With: Unity
  • Platform: Web & Desktop
  • Team: Coral Coders
